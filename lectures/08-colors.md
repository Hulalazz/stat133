### Naming Colors

- Color names in English with `colors()`
- Colors in hexadecimal notation
- Color model spaces:
	+ RGB with `rgb()`
	+ HSV with `hsv()`
	+ HCL with `hcl()`
	+ etc
- Color palettes:
	+ `rainbow()`
	+ `heat.colors()`
	+ `topo.colors()`
	+ `terrain.colors()`
	+ `cm.colors()`
- R package `"RColorBrewer"
